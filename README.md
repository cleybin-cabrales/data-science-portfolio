## Projects

### Random Forest - Drug Response Prediction
Trained and compared Decision Tree and Random Forest classifiers on patient data. Analyzed feature importance across blood pressure, cholesterol, and age. Achieved 61% accuracy with clear improvement in stability over the single-tree baseline.
Tools: Python, Scikit-learn, Pandas, Matplotlib, Seaborn

### Logistic Regression - Customer Churn Prediction
Built a binary classification model on a 200-observation, 28-variable churn dataset. Covered the full pipeline from assumption validation and feature selection to evaluation with precision, recall, and F1-score.
Tools: Python, Scikit-learn, Pandas, Seaborn

### K-Means Clustering - Customer Segmentation
Applied unsupervised clustering to a financial customer dataset. Handled missing values with median imputation, scaled features with StandardScaler, and assessed cluster quality using silhouette score.
Tools: Python, Scikit-learn, Pandas, Matplotlib

### DNN Linear Regression - Housing Price Prediction
Built a Deep Neural Network regression model using TensorFlow and Keras to predict housing prices. Applied StandardScaler, train/test split, and evaluated results with MAE, MSE, and R2.
Tools: Python, TensorFlow, Keras, Pandas, Scikit-learn
